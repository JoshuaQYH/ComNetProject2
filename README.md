## 二、实验环境
 IDE： Visual stdio 2017
 编程语言： C++

 ## 三、实验步骤分解
 1. 设计网络拓扑图，定义节点之间的连接代价。
 2. 定义通信服务类，确定接收和发送消息。
 3. 报文协议设计，更新交换路由信息，发送消息报文，回显报文。同时具备解码功能和编码功能，用于网路传输。
 4. 封装路由选择类，RIP和OSPF，利用两种路由选择算法LS & DV来实现路由选择。
