## 二、实验环境
 IDE： Visual stdio 2017
 编程语言： C++
 
 ## 三、实验步骤分解
 1. 设计网络拓扑图，定义节点之间的连接代价。
 2. 每一个节点既作为客户端和其它节点通信，又作为路由器跟其他节点周期性地交换路由信息。
 3. 每一个节点有接收端口和发送端口。
 4. 利用两种路由选择算法LS & DV来实现路由选择。
